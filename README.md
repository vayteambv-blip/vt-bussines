# VT Business

Локальный проект для управления строительными объектами в Бельгии.

## Подход

Начинаем максимально просто: сначала бизнес-контекст, потом функциональная карта, потом план, потом код.

1. Сначала бизнес-контекст.
2. Потом функционал.
3. Потом план.
4. Потом код.

## Структура

```text
VT-bussines/
  AGENTS.md
  README.md
  spec/
    01-бизнес-контекст.md
    02-функциональная-карта.md
```

## Главный файл

- `spec/01-бизнес-контекст.md` - смысл проекта, бизнес-контекст и следующий раздел для функционала.
- `spec/02-функциональная-карта.md` - что пользователь делает, видит и планирует.
